---
title: "productsByDomainV2"
description: "Returns an array of Products filtered by a given store domain. This query can be used for paging through a set of products from a domain from Shopify."
---

<Note>
  For example, if the store's canonical URL is [https://abc.myshopify.com](https://abc.myshopify.com), then the domain is abc.myshopify.com, all lowercased.

  As the number of products may be significantly large, it's recommended to paginate through the results.
</Note>

<br />

<Note>
  [AmazonProduct](/reference/amazonproduct) and [ShopifyProduct](/reference/shopifyproduct) returned by this query has similaries and differences that could be found in [the spreadsheet](/docs/amazon-and-shopify-product-comparison)
</Note>

---

### Arguments

<ParamField path="`input`" type="ProductsByDomainInput!">
  An object containing the domain name of the store. The search is case-insensitive.
</ParamField>
<ParamField path="`pagination``" type="OffsetPaginationInput!">
  An object containing pagination parameters.
</ParamField>
---

### Returns

<ParamField path="Product" type="Product">
  Any requested field from an object which implements the [Product](https://rye-docs.readme.io/reference/product) interface.
</ParamField>

<Note>The query returns an empty array if no products are found</Note>

---

### Examples

```GraphQL GraphQL
query DemoShopifyProductByDomain {
  productsByDomainV2(
    input: { domain: "hiutdenim.co.uk" }
    pagination: { limit: 3, offset: 2 }
  ) {
    id
    title
    marketplace
    description
    isAvailable
    price {
      ... on Price {
        value
        currency
        displayValue
      }
    }
  }
}
```
