---
title: "updateCartSelectedShippingOptions"
description: "Updates shipping methods for stores in the cart"
---

> ðŸ“˜
> 
> Selected shipping options must be provided either by using this mutation, or they should be passed to [RyePay](/docs/rye-pay).submit method or, if you use 'backend only' payment flow, should be passed to [submitCart](/reference/submitcart) mutation directly. If not provided, cart submission will fail with `SUBMIT_STORE_FAILED` error.

---

### Arguments

`input` : [UpdateCartSelectedShippingOptionsInput!](/reference/updatecartselectedshippingoptionsinput)

The `input` object contains the unique identifier (ID) for the cart and an array of shipping options associated with a store the shipping belongs to

> ðŸ“˜
> 
> [SelectedShippingOption.store](/reference/selectedshippingoption#fields)must be the same as as [AmazonStore.store](/reference/amazonstore#fields) or [ShopifyStore.store](/reference/shopifystore#fields) fields

---

### Returns

[CartResponse.\*](/reference/cartresponse#fields): [CartResponse](/reference/cartresponse)

Any requested field from the [CartResponse](/reference/cartresponse) object.

### Errors

If the cart is not found then a corresponding error is returned

JSON

`{
    "errors": [
        {
            "message": "Cart not found: someInvalidCartId",
            "path": [
                "getCart"
            ]
        }
    ],
    "data": null
}
`

If the provided store is not present in the cart then `STORE_NOT_FOUND`error is returned

JSON

`{
    "data": {
        "updateCartSelectedShippingOptions": {
            "cart": {
                "id": "0OSTHkAAhrGxr9nBN33c",
                "stores": [
                    {
                        "errors": [],
                        "store": "rye-test-store.myshopify.com",
                        "cartLines": [
                            {
                                "quantity": 1,
                                "variant": {
                                    "id": "39943631962199"
                                }
                            }
                        ]
                    }
                ]
            },
            "errors": [
                {
                    "code": "STORE_NOT_FOUND",
                    "message": "store NonExistingStore not found"
                }
            ]
        }
    }
}
`
