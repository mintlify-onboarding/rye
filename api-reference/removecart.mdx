---
title: "removeCart"
description: "Removes an existing shopping cart based on the provided cart ID. It allows users to remove a cart that is no longer needed or relevant."
---

<Warning> The cart will be removed regardless if the products are present in the cart or not and regardless cart submission status.</Warning>

---

### Arguments

<ParamField path="input" type="CartDeleteInput!">
  The `input` object contains the unique identifier (ID) for the cart that needs to be deleted.
</ParamField>

---

### Returns

<ParamField path="DeletedCart.deletedId" type="DeletedCart">
  The unique identifier (ID) of the deleted cart.
</ParamField>

---

### Examples

```json Query arguments
{
    "input": {
        "id": "{{cartId}}"
    }
}
```


```GraphQL GraphQL
mutation ($input: CartDeleteInput!) {
    removeCart(input: $input) {
        deletedId
    }
}
```

<ResponseExample>

```json Response
{
    "data": {
        "removeCart": {
            "deletedId": "teWvPufPy8c2AcfkfBo9"
        }
    }
}
```
</ResponseExample>

---

### Errors

If the cart is not found then a corresponding error is returned


```JSON JSON
{
    "errors": [
        {
            "message": "Cart not found: someInvalidCartId",
            "path": [
                "getCart"
            ]
        }
    ],
    "data": null
}
```
