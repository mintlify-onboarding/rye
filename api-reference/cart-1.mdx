---
title: "getCart"
description: "Retrieve a Cart by its ID. This query can be used to retrieve information about an existing cart."
---

### Arguments

`id`: `ID!`

The ID of the [Cart](https://rye-docs.readme.io/reference/cart-2)

---

### Returns

[CartResponse.\*](/reference/cartresponse#fields): [CartResponse](/reference/cartresponse)

Any requested field from the [CartResponse](/reference/cartresponse) object.

---

### Errors

If the cart is not found then a corresponding error is returned

JSON

`{
    "errors": [
        {
            "message": "Cart not found: someInvalidCartId",
            "path": [
                "getCart"
            ]
        }
    ],
    "data": null
}
`
